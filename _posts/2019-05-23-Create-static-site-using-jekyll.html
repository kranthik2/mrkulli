Jekyll is a Ruby Gem and a simple, blog-aware, static site generator that can be installed on most systems.

how to generate site using Jekyll
  1. Install a full Ruby development environment https://jekyllrb.com/docs/installation/
  2. Install Jekyll and bundler gems
    3. {% highlight shell %} gem install jekyll bundler {% endhighlight %}
  3. Create a new Jekyll site
    4. {% highlight shell %} jekyll new someblog {% endhighlight %}
  4. Change into your new directory
    5. {% highlight shell %} cd someblog {% endhighlight %}
  5. Build the site by running below command
    6. {% highlight shell %} bundle exec jekyll build {% endhighlight %}
  6. Run server and make it available at localserver
    7. {% highlight shell %} bundle exec jekyll serve {% endhighlight %}

    go to http://localhost:4000 url and access your site
